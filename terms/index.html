2. MyChessOpening Services

The main purpose of this site is to provide a useful analysis of the chess openings a certain player has been playing recently on its chess games at chess.com or lichess.org

This analysis will tell the user which openings are working well (making it win the games), and what opening it should avoid playing.

MyChessOpening DOES NOT have any association with any of these two websites, it has been created by users of these sites.

2.1 The data for the analysis

The data used to offer such analysis is the most recent chess games of the chosen player.

All data used to perform that analysis is public, acessible for anyone on the internet, I've just created a program to gather all of them automatically and apply an algorithm to interpret that data.

They are obtained using the lichess.org API and chess.com API

If a chess player has created an account in any of these sites, it has automatically agreed to share his games on the internet so that anyone can gather and use it as they want.

2.2 The results of the analysis

After an analysis is finished, its results are saved in MyChessOpening's database and an unique analysis id is generated allowing the results to be shared with anyone on the internet.

In addition to the results, the name of the user who requested the analysis and the date and time that the analysis was requested are also saved and displayed on the results page.
